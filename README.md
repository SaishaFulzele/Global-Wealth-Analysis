# 🌍 Global Wealth Opportunity & Income Simulation Engine
---

## 🚀 Why This Project Matters

Choosing the right country can significantly impact long-term wealth accumulation.

However, most people make this decision based only on salary — ignoring critical factors like taxation, cost of living, and rent.

This project solves that problem by combining these variables into a data-driven model that evaluates:

👉 Where can you actually build wealth?

Using real-world data and scenario simulations (optimistic, base, pessimistic), this project provides a more realistic view of global financial opportunities.

---
## 💡 Key Highlight

Built a scenario-based financial simulation model across 30 countries to evaluate global wealth opportunities under different economic conditions.

---

## 📌 Overview
This project analyzes wealth-building opportunities across 30 countries using real-world data.

It evaluates:
- Salary levels
- Tax rates
- Cost of living
- Rent

to determine where individuals can maximize disposable income and long-term wealth.

---

## ⚙️ Features
- 📊 Opportunity Score Model
- 🌎 Country Comparison Engine
- 📉 Scenario Analysis (Optimistic, Base, Pessimistic)
- 📈 Wealth Simulation
- 🔥 Visualizations (Heatmaps, Scatter Plots, Growth Curves)
- 🌐 Integrated real-world economic data (World Bank)
---

## 🧠 Key Insights

### 💰 Wealth Leaders
Luxembourg, Singapore, and Ireland consistently rank at the top across all scenarios.

### ⚠️ Risk Impact
Disposable income drops significantly under pessimistic scenarios, even in strong economies.

### ⚖️ High Risk–High Reward Economies
Countries like Australia and UAE show strong upside but higher volatility depending on economic conditions.

---

## 🧾 Conclusion
Choosing the right country is not just about salary — it's about balancing income, costs, and economic stability.

Scenario-based analysis provides a more realistic view of future financial outcomes.
